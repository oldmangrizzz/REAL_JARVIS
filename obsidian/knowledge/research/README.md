## Why this exists

The wiki's `concepts/` pages describe the **current** doctrine (NLB, Aragorn-Class,
TinCan-Firewall, etc.). This `research/` tree contains the **source** material those
concepts were distilled from — the whitepapers, operator-class deployment guides,
digital-person blueprints, identity system-prompts, and operator-mind session logs
that pre-date the codebase.

Together they form the **evidence corpus**: every canonical claim in this project
traces back to a document here.
