# Headphone Compatibility Matrix

## H1/H2 Chip Headphones (Apple/Empire):

| Feature | AirPods | AirPods Pro | Beats H1/H2 |
|---------|---------|-------------|-------------|
| BT Pairing to Watch | ✅ Yes | ✅ Yes | ✅ Yes |
| A2DP Playback | ✅ Yes | ✅ Yes | ✅ Yes |
| Mic Passthrough | ✅ Yes | ✅ Yes | ✅ Yes |
| Spatial Audio | ✅ Yes | ✅ Yes | ✅ Yes |
| Mic to Watch | ✅ Yes | ✅ Yes | ✅ Yes |
| Microphone Access | ✅ Yes | ✅ Yes | ✅ Yes |

| Mic to Watch | ❌ No | ⚠️ Future (LC3) |
| Microphone Access | ❌ No | ⚠️ Future (LC3) |
