## How these pages are used

1. **Identity wiring** — each character page declares the canonical traits the
   matching [[research/identities/]] system-prompt leans on.
2. **MCU context** — [[concepts/MCU]] ingest references these pages when
   weighting biographical-mass retrieval.
3. **Authorship audit** — keeps the operator's creative sources separable from
   their original invention (GMRI, Aragorn-Class, NLB, TinCan-Firewall).

## Style rules

- Summaries ≤ ~400 words per character.
- Always cite at least one web source + one local PDF / research file.
- Distinguish **comics (Earth-616) vs MCU / DCEU** canon explicitly.
- Never copy prose verbatim from Wikipedia / Fandom. Paraphrase.
- Cross-link to `concepts/` and `research/` liberally.
