MEMORY ARCHITECTURE The memory stack is a MemGPT and Cognee hybrid implemented by MemoryEngine. Main context acts as active RAM with system instructions, a working context dictionary, and an eight-slot FIFO queue. External context is a persisted knowledge graph on disk. Memify ingests execution traces, recursive thoughts, stigmergic signals, vagal tone, node registry records, and trace files; chunks them semantically; creates document, chunk, and entity nodes; and connects them with contains, episode, and mentions relations. Retrieval is vector style page-in: the engine embeds the query into a fixed 32-slot feature vector, ranks nodes by cosine similarity, returns the best matches, and logs the page fault into recursive_thoughts. Somatic reflex edges are recorded separately and then surfaced during page-in.
